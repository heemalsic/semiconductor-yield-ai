# semiconductor-yield-ai

Starter project for semiconductor yield forecasting, anomaly detection, and SPC analysis.

## Structure

- `data/` — CSV dataset(s)
- `src/` — preprocessing, modeling, analysis modules
- `dashboard/` — Streamlit dashboard starter
- `notebooks/` — exploration notebooks

## Quick start

Install requirements:

```powershell
pip install -r requirements.txt
```

Run the dashboard (Streamlit):

```powershell
cd dashboard; streamlit run app.py
```

Run the notebook with Jupyter or an editor that supports notebooks.
